# OTP Авторизация по номеру телефона

Форма авторизации с вводом номера телефона и OTP-кода, реализованная на React.

## Функциональность

- Ввод номера телефона
- Маска телефона: +7 XXX XXX XX XX
- Разрешён ввод только цифр
- Валидация количества символов
- Отправка OTP-кода
- Таймер повторной отправки кода
- Повторная отправка OTP
- Ввод 6-значного кода
- Валидация OTP
- Авторизация пользователя
- Получение сессии пользователя
- Отображение данных пользователя после входа
- Mock API (имитация backend)


## Используемые технологии

- React

- CSS
- Vite
- Fetch API (mock)

## Логика работы

### 1. Ввод телефона
Пользователь вводит номер телефона в формате:
**+7 913 123 45 67**

После нажатия "Продолжить" отправляется запрос на создание OTP-кода.

### 2. Получение OTP

API возвращает:
{
success: true,
retryDelay: 40000
}


Запускается таймер повторной отправки кода.


### 3. Ввод OTP

Пользователь вводит 6-значный код.

Происходит запрос авторизации:
{
phone,
code
}

### 4. Авторизация

API возвращает:

{
success: true,
user: {...},
token: "string"
}

Токен сохраняется в состоянии приложения.


### 5. Получение сессии

После авторизации выполняется запрос:
GET /session
И отображаются данные пользователя.


## Mock API

В проекте используется mock API:

- requestOtp(phone)
- loginWithOtp(phone, code)
- getUserSession(token)

Файл:

src/api/authApi.js

При наличии реального backend достаточно заменить содержимое этого файла.
